Como obter a API Key do Brevo
  1. Acesse app.brevo.com e crie uma conta gratuita (até 300 e-mails/dia).
  2. Clique no seu nome → SMTP & API → aba API Keys.
  3. Clique em Create a new API key, copie e cole acima.
  4. Em Senders → Add a new sender verifique o e-mail remetente.
Como obter a API Key do Resend
  1. Acesse resend.com e crie uma conta (3.000 e-mails/mês grátis).
  2. Vá em API Keys → Create API Key.
  3. Em Domains, adicione e verifique seu domínio de envio.
  4. Cole a API Key acima e use um e-mail do domínio verificado como remetente.

Como Configurar cada Provedor

Gratuito: até 300 e-mails/dia e 9.000/mês. Sem necessidade de configurar servidor SMTP.
1
Acesse app.brevo.com e crie uma conta gratuita.
2
No menu superior, clique no seu nome → SMTP & API.
3
Vá em API Keys → Create a new API key.
4
Copie a chave gerada e cole no campo API Key da aba Provedor de E-mail.
5
Em Senders → Add a new sender, verifique o e-mail que usará como remetente.
Endpoint: https://api.brevo.com/v3/smtp/email
Auth:     Header → api-key: SUA_API_KEY
Grátis:   300 e-mails/dia · 9.000/mês · sem cartão de crédito
Gratuito: até 3.000 e-mails/mês no plano free. API moderna com alta entregabilidade.
1
Acesse resend.com e crie uma conta.
2
Vá em API Keys → Create API Key e copie a chave.
3
Em Domains, adicione e verifique o seu domínio de envio (verificação via DNS).
4
Use um e-mail do domínio verificado no campo E-mail Remetente.
Endpoint: https://api.resend.com/emails
Auth:     Header → Authorization: Bearer SUA_API_KEY
Grátis:   3.000 e-mails/mês · 1 domínio personalizado
SMTP é um fallback opcional. Para produção em condomínios, prefira Brevo ou Resend — sem bloqueio de porta 587 por ISP.
