===========================================
REGISTRO DE ERROS - SISTEMA SERRA DA LIBERDADE
===========================================

Este arquivo serve para registrar erros e problemas identificados no sistema.

-------------------------------------------
FORMATO DO REGISTRO
-------------------------------------------

[DATA/HORA] - [TIPO DE ERRO] - [DESCRIÇÃO]
Detalhes adicionais...

-------------------------------------------
TIPOS DE ERRO
-------------------------------------------

- ERRO_500: Internal Server Error
- ERRO_404: Página não encontrada
- ERRO_403: Acesso negado
- ERRO_LOGIN: Problemas no login
- ERRO_SESSAO: Problemas de sessão
- ERRO_API: Problemas nas APIs
- ERRO_CAMINHO: Caminhos incorretos
- ERRO_JS: Erros JavaScript
- ERRO_PHP: Erros PHP

-------------------------------------------
INSTRUÇÕES PARA REGISTRO
-------------------------------------------

1. Ao identificar um erro, registre aqui com data, hora e descrição
2. Inclua o máximo de detalhes possível (arquivo, linha, mensagem)
3. Após corrigir, mova para "HISTÓRICO DE CORREÇÕES"
4. Use a página debug_erros.html para diagnóstico detalhado
