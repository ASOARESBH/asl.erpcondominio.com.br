Como Configurar

Brevo
Resend
SMTP Personalizado
Gratuito: até 300 e-mails/dia e 9.000/mês. Sem necessidade de configurar SMTP.
1
Acesse app.brevo.com e crie uma conta gratuita.
2
No menu superior, clique no seu nome → SMTP & API.
3
Vá em API Keys → Create a new API key.
4
Copie a chave gerada e cole no campo API Key acima.
5
Configure E-mail Remetente com um endereço verificado no Brevo (Senders → Add a new sender).
Endpoint: https://api.brevo.com/v3/smtp/email
Auth: Header: api-key: SUA_API_KEY
Grátis: 300 e-mails/dia · 9.000/mês · sem cartão
Gratuito: até 3.000 e-mails/mês no plano free. API moderna com alta entregabilidade.
1
Acesse resend.com e crie uma conta.
2
Vá em API Keys → Create API Key.
3
Adicione e verifique seu domínio em Domains.
4
Configure o E-mail Remetente usando um endereço do domínio verificado.
Endpoint: https://api.resend.com/emails
Auth: Header: Authorization: Bearer SUA_API_KEY
Grátis: 3.000 e-mails/mês · 1 domínio personalizado
SMTP é um fallback opcional. Para produção, prefira Brevo ou Resend.
